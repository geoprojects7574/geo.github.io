Independencia y estacionariedad

Procesos estocásticos en RL y robótica

Aplicaciones en aprendizaje reforzado y robótica

Idea central del trabajo

Los tres procesos elegidos son:

  1. ruido blanco gaussiano,
  2. proceso autorregresivo de orden 1, AR(1),
  3. caminata aleatoria.

Conceptos previos

¿Qué significa independencia?

Un proceso estocástico \(\{X_t\}\) es independiente si para cualquier conjunto finito de tiempos \(t_1,t_2,\dots,t_n\) se cumple:

\[ F_{X_{t_1},X_{t_2},\dots,X_{t_n}}(x_1,x_2,\dots,x_n) = \prod_{i=1}^n F_{X_{t_i}}(x_i) \]

Si existe densidad conjunta, esto equivale a:

\[ f_{X_{t_1},X_{t_2},\dots,X_{t_n}}(x_1,x_2,\dots,x_n) = \prod_{i=1}^n f_{X_{t_i}}(x_i) \]

Interpretación sencilla

Si el proceso es independiente, entonces conocer lo que ocurrió en un instante anterior no cambia la distribución de lo que pasa después.

En términos condicionales, esto se interpreta como:

\[ P(X_t \mid X_{t-1}) = P(X_t) \]

Si esta igualdad no se cumple, entonces el proceso no es independiente.

Conceptos previos

¿Qué significa estacionariedad?

Estacionariedad estricta

El proceso es estrictamente estacionario si, para cualquier desplazamiento temporal \(h\),

\[ (X_{t_1},X_{t_2},\dots,X_{t_n}) \overset{d}{=} (X_{t_1+h},X_{t_2+h},\dots,X_{t_n+h}) \]

Esto significa que la distribución conjunta no cambia cuando el tiempo se desplaza.

Estacionariedad débil

Un proceso es débilmente estacionario si cumple tres condiciones:

\[ E[X_t] = \mu \quad \text{para todo } t \]

\[ \operatorname{Var}(X_t) = \sigma^2 \quad \text{para todo } t \]

\[ \operatorname{Cov}(X_t, X_{t+h}) = \gamma(h) \]

Es decir:

  • la media no cambia con el tiempo,
  • la varianza no cambia con el tiempo,
  • la covarianza depende solo del rezago \(h\), no del instante exacto.

Proceso 1: Ruido blanco gaussiano

Situación real

Supongamos un robot móvil con un sensor de distancia.
Cada vez que mide la distancia a una pared, comete un pequeño error aleatorio debido al ruido electrónico interno del sensor.

Si ese error no depende del error anterior y además tiene siempre la misma distribución, podemos modelarlo como:

\[ X_t \sim iid\ \mathcal{N}(0,\sigma^2) \]

Aquí:

  • \(X_t\) representa el error del sensor en el instante \(t\),
  • la media es 0 porque el sensor no está sesgado,
  • la varianza \(\sigma^2\) representa la intensidad del ruido.

Independencia

Queremos probar si el proceso es independiente.

Por definición, las variables \(X_t\) son iid.
Entonces, para cualquier colección de tiempos \(t_1,\dots,t_n\),

\[ f_{X_{t_1},\dots,X_{t_n}}(x_1,\dots,x_n) = \prod_{i=1}^n f_{X_{t_i}}(x_i) \]

Esto significa que la densidad conjunta se puede escribir como producto de densidades marginales.

Por lo tanto, el proceso cumple exactamente la definición de independencia.

Conclusión

\[ \{X_t\} \text{ es un proceso independiente} \]

Estacionariedad

Ahora verificamos la estacionariedad débil.

Paso 1: Media

Como \(X_t \sim \mathcal{N}(0,\sigma^2)\), entonces:

\[ E[X_t] = 0 \]

Esto vale para todo \(t\).

Entonces la media es constante.

Paso 2: Varianza

También, por la distribución normal,

\[ \operatorname{Var}(X_t) = \sigma^2 \]

Esto también vale para todo \(t\).

Entonces la varianza es constante.

Estacionariedad

Paso 3: Covarianza

Si \(h=0\), entonces:

\[ \operatorname{Cov}(X_t,X_t)=\operatorname{Var}(X_t)=\sigma^2 \]

Si \(h\neq 0\), como \(X_t\) y \(X_{t+h}\) son independientes, se tiene:

\[ \operatorname{Cov}(X_t,X_{t+h})=0 \]

Por tanto, la función de autocovarianza es:

\[ \gamma(h)= \begin{cases} \sigma^2, & h=0 \\ 0, & h\neq 0 \end{cases} \]

Esta depende solo del rezago \(h\).

Conclusión

El proceso es débilmente estacionario.

Además, como todas las variables tienen la misma distribución y son independientes, también es estrictamente estacionario.

Conclusión final

  • Independencia: sí
  • Estacionariedad débil: sí
  • Estacionariedad estricta: sí

Interpretación real

Este modelo representa bien el ruido puro de un sensor cuando no existe memoria entre errores consecutivos.

Proceso 2: Proceso AR(1)

Situación real

Ahora pensemos en un robot que intenta mantener una trayectoria recta.

Si en un instante se desvió un poco hacia la derecha, es probable que en el siguiente instante todavía conserve parte de ese desvío. Es decir, el error actual no es completamente nuevo: arrastra parte del error anterior.

Este comportamiento puede modelarse con:

\[ X_t = \phi X_{t-1} + \varepsilon_t \]

donde:

  • \(X_t\) es el error de trayectoria en el instante \(t\),
  • \(\phi\) mide cuánto del error pasado persiste,
  • \(\varepsilon_t\) es un ruido aleatorio nuevo.

Definición

Se considera el proceso:

\[ X_t = \phi X_{t-1} + \varepsilon_t \]

con

\[ |\phi| < 1 \]

y

\[ \varepsilon_t \sim iid\ \mathcal{N}(0,\sigma_\varepsilon^2) \]

Aquí, \(\varepsilon_t\) es ruido blanco.

Independencia

Queremos saber si \(X_t\) es independiente de \(X_{t-1}\).

Partimos de la definición del proceso:

\[ X_t = \phi X_{t-1} + \varepsilon_t \]

Si condicionamos en \(X_{t-1}\), entonces el único término aleatorio es \(\varepsilon_t\), así que:

\[ X_t \mid X_{t-1} \sim \mathcal{N}(\phi X_{t-1}, \sigma_\varepsilon^2) \]

Observa que la media condicional depende de \(X_{t-1}\).

Eso significa que la distribución de \(X_t\) cambia cuando conocemos el valor anterior.

Por tanto,

\[ P(X_t \mid X_{t-1}) \neq P(X_t) \]

Conclusión

El proceso no es independiente.

Otra forma de ver la dependencia

También puede verse usando covarianza.

Sea

\[ \gamma(1)=\operatorname{Cov}(X_t,X_{t-1}) \]

Como luego demostraremos,

\[ \gamma(1)=\phi \gamma(0) \]

Si \(\phi \neq 0\), esta covarianza es distinta de cero.

Entonces:

\[ \operatorname{Cov}(X_t,X_{t-1}) \neq 0 \]

y eso implica dependencia.

Estacionariedad

Paso 1: Media

Tomamos esperanza en ambos lados de:

\[ X_t = \phi X_{t-1} + \varepsilon_t \]

Entonces:

\[ E[X_t] = \phi E[X_{t-1}] + E[\varepsilon_t] \]

Como el ruido tiene media cero,

\[ E[\varepsilon_t]=0 \]

obtenemos:

\[ E[X_t] = \phi E[X_{t-1}] \]

Estacionariedad

Si el proceso es estacionario, la media debe ser constante. Denotémosla por \(\mu\):

\[ \mu = \phi \mu \]

Entonces:

\[ (1-\phi)\mu = 0 \]

Como \(|\phi|<1\), se tiene \(\phi \neq 1\), así que:

\[ \mu = 0 \]

La media estacionaria es constante e igual a cero.

Estacionariedad

Paso 2: Varianza

Partimos de:

\[ X_t = \phi X_{t-1} + \varepsilon_t \]

Calculamos varianza:

\[ \operatorname{Var}(X_t) = \operatorname{Var}(\phi X_{t-1} + \varepsilon_t) \]

Como \(\varepsilon_t\) es independiente de \(X_{t-1}\),

\[ \operatorname{Var}(X_t) = \phi^2 \operatorname{Var}(X_{t-1}) + \operatorname{Var}(\varepsilon_t) \]

o sea,

\[ \operatorname{Var}(X_t) = \phi^2 \operatorname{Var}(X_{t-1}) + \sigma_\varepsilon^2 \]

Si existe estacionariedad, entonces la varianza es constante, digamos \(\sigma_X^2\):

\[ \sigma_X^2 = \phi^2 \sigma_X^2 + \sigma_\varepsilon^2 \]

Estacionariedad

Despejamos:

\[ \sigma_X^2 - \phi^2 \sigma_X^2 = \sigma_\varepsilon^2 \]

\[ \sigma_X^2(1-\phi^2)=\sigma_\varepsilon^2 \]

\[ \sigma_X^2=\frac{\sigma_\varepsilon^2}{1-\phi^2} \]

Esta varianza es finita si y solo si:

\[ |\phi|<1 \]

Entonces, bajo esa condición, la varianza es constante.

Estacionariedad

Paso 3: Autocovarianza

Definimos:

\[ \gamma(h)=\operatorname{Cov}(X_t,X_{t-h}) \]

Para \(h=1\):

\[ \gamma(1)=\operatorname{Cov}(X_t,X_{t-1}) \]

Usamos la ecuación del proceso:

\[ X_t=\phi X_{t-1}+\varepsilon_t \]

Entonces:

\[ \gamma(1) = \operatorname{Cov}(\phi X_{t-1}+\varepsilon_t, X_{t-1}) \]

Estacionariedad

Por linealidad de la covarianza:

\[ \gamma(1) = \phi \operatorname{Cov}(X_{t-1},X_{t-1}) + \operatorname{Cov}(\varepsilon_t, X_{t-1}) \]

Como \(\varepsilon_t\) es independiente de \(X_{t-1}\):

\[ \operatorname{Cov}(\varepsilon_t, X_{t-1}) = 0 \]

y como

\[ \operatorname{Cov}(X_{t-1},X_{t-1}) = \operatorname{Var}(X_{t-1})=\gamma(0) \]

obtenemos:

\[ \gamma(1)=\phi\gamma(0) \]

Estacionariedad

Repitiendo el razonamiento de forma recursiva se obtiene:

\[ \gamma(h)=\phi^h \gamma(0), \qquad h\geq 0 \]

Como

\[ \gamma(0)=\sigma_X^2=\frac{\sigma_\varepsilon^2}{1-\phi^2} \]

entonces:

\[ \gamma(h)=\phi^h \frac{\sigma_\varepsilon^2}{1-\phi^2} \]

Esta cantidad depende solo del rezago \(h\).

Conclusión

El proceso AR(1) es débilmente estacionario siempre que:

\[ |\phi|<1 \]

Conclusión final

  • Independencia: no
  • Estacionariedad débil: sí, si \(|\phi|<1\)

Interpretación real

En robótica, este modelo representa un error con memoria: el pasado influye, pero su efecto se va amortiguando si \(|\phi|<1\).

En aprendizaje reforzado, se parece a situaciones donde el error o estado no cambia de forma totalmente nueva en cada paso, sino que arrastra parte de la dinámica previa.

Proceso 3: Caminata aleatoria

Situación real

Imagina un robot que avanza sobre el piso y en cada paso acumula un pequeño error de posición.

Ese error no se corrige, sino que se suma al anterior. Entonces la posición estimada va derivando con el tiempo.

Ese comportamiento se modela como:

\[ X_t = X_{t-1} + \varepsilon_t \]

Aquí:

  • \(X_t\) puede ser el error acumulado de posición,
  • \(\varepsilon_t\) es el pequeño error nuevo en cada paso.

Definición

Se considera el proceso:

\[ X_t = X_{t-1} + \varepsilon_t \]

con

\[ \varepsilon_t \sim iid\ \mathcal{N}(0,\sigma^2) \]

Equivalentemente, si expandimos recursivamente:

\[ X_t = X_0 + \sum_{i=1}^t \varepsilon_i \]

Esto muestra claramente que \(X_t\) acumula todos los errores pasados.

Independencia

Partimos de:

\[ X_t = X_{t-1} + \varepsilon_t \]

Si condicionamos en \(X_{t-1}\), entonces:

\[ X_t \mid X_{t-1} \sim \mathcal{N}(X_{t-1},\sigma^2) \]

Aquí la media condicional es \(X_{t-1}\).

Eso significa que saber el valor anterior cambia por completo la distribución del valor actual.

Por tanto:

\[ P(X_t \mid X_{t-1}) \neq P(X_t) \]

Conclusión

La caminata aleatoria no es independiente.

Estacionariedad

Paso 1: Media

Usamos la forma expandida:

\[ X_t = X_0 + \sum_{i=1}^t \varepsilon_i \]

Tomando esperanza:

\[ E[X_t] = E[X_0] + \sum_{i=1}^t E[\varepsilon_i] \]

Estacionariedad

Como cada ruido tiene media cero,

\[ E[\varepsilon_i]=0 \]

entonces:

\[ E[X_t]=E[X_0] \]

Si denotamos:

\[ E[X_0]=\mu_0 \]

queda:

\[ E[X_t]=\mu_0 \]

La media puede ser constante.

Estacionariedad

Paso 2: Varianza

Partimos otra vez de:

\[ X_t = X_0 + \sum_{i=1}^t \varepsilon_i \]

Suponiendo que \(X_0\) es constante o independiente del ruido, se tiene:

\[ \operatorname{Var}(X_t) = \operatorname{Var}\left(\sum_{i=1}^t \varepsilon_i\right) \]

Estacionariedad

Como los \(\varepsilon_i\) son independientes:

\[ \operatorname{Var}(X_t) = \sum_{i=1}^t \operatorname{Var}(\varepsilon_i) \]

y como cada uno tiene varianza \(\sigma^2\):

\[ \operatorname{Var}(X_t)=t\sigma^2 \]

Esta varianza crece con el tiempo.

Por tanto, no es constante.

Entonces el proceso no es estacionario.

Estacionariedad

Paso 3: Autocovarianza

Tomemos dos tiempos con \(s<t\).

Entonces:

\[ X_s = X_0 + \sum_{i=1}^s \varepsilon_i \]

\[ X_t = X_0 + \sum_{i=1}^t \varepsilon_i \]

Para simplificar, supongamos \(X_0=0\). Entonces:

\[ X_s = \sum_{i=1}^s \varepsilon_i, \qquad X_t = \sum_{i=1}^t \varepsilon_i \]

La covarianza es:

\[ \operatorname{Cov}(X_s,X_t) = \operatorname{Cov}\left(\sum_{i=1}^s \varepsilon_i,\sum_{j=1}^t \varepsilon_j\right) \]

Estacionariedad

Por independencia de los incrementos, solo sobreviven los términos comunes entre ambas sumas, es decir, los primeros \(s\) ruidos.

Entonces:

\[ \operatorname{Cov}(X_s,X_t)=s\sigma^2 \]

Observa que esta expresión depende de \(s\), no solo del rezago \(t-s\).

Por lo tanto, la autocovarianza no depende únicamente del rezago.

Eso confirma nuevamente que la caminata aleatoria no es débilmente estacionaria.

Conclusión final

  • Independencia: no
  • Estacionariedad débil: no

Interpretación real

Este modelo representa muy bien la acumulación de error en posición, rumbo o estimación. Aunque cada perturbación nueva sea pequeña, la suma de muchas perturbaciones hace que la incertidumbre crezca con el tiempo.

Aplicación en aprendizaje reforzado

¿Dónde aparece esto en RL?

Estos procesos ayudan a modelar fenómenos reales como:

  • ruido en sensores,
  • perturbaciones de actuadores,
  • error de estimación del estado,
  • deriva acumulada en la localización,
  • correlación temporal en la dinámica del entorno.

Ejemplos

  • Ruido blanco: error instantáneo del sensor.
  • AR(1): error del control con persistencia temporal.
  • Caminata aleatoria: desviación acumulada de posición.
